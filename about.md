Я — Василь. Senior Software Engineer з 15+ роками досвіду в побудові масштабованих, event-driven і cloud-native систем у сферах healthcare, video conferencing, messaging і фінансів. Спеціалізуюсь на backend-архітектурі — надійній, підтримуваній і оптимізованій під навантаження.

Останні чотири роки працював у healthcare (CtheSigns) — розробляв мікросервіси та event-driven рішення на Java/Kotlin: REST API (для вебзастосунку та AWS Lambda), RSocket контролери (міжсервісна комунікація), WebSocket обробники (десктопний клієнт). Переробив і оптимізував процес синхронізації даних пацієнтів — зробив сервіс масштабованим і стійким під навантаженням.

Стек: MongoDB, Redis, AWS (ECS, SQS, SNS, Lambda, CloudWatch, ElastiCache), Docker, Terraform/Terragrunt, GitHub Actions. Налагоджував observability через JMX, CloudWatch, ELK, Percona, NewRelic. Розробляв загальнокорпоративні policy (Data Integrity) та Architecture Decision Records (ADR).

Нещодавно зайшов як Tech Lead у SaaS ERP проєкт — провів повну операційну трансформацію: декомпозиція монолітного Spring Boot бекенду на мікросервіси з async обробкою через SQS, оптимізація звітів, кешування, request handling. Побудував CI/CD пайплайни для всіх платформ — Web, iOS, macOS, Android, Windows — включно з code signing, notarization і дистрибуцією в App Store, Google Play і S3.

З нуля побудував весь QA процес: тест-стратегія, smoke тести на всіх платформах, E2E автоматизація на Selenium + Cucumber, інтеграція з BrowserStack, Slack-нотифікації. Запровадив інженерні процеси — severity рівні інцидентів, structured bug reporting, release coordination.

Подобається тримати в голові повну картину — від коду до інфраструктури до процесів команди. І перетворювати хаос на щось передбачуване.

## А ще — FPGA

Цей блог — щоденник того, що з цього виходить.
